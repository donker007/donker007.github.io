What is machine learning?

发表于 2016-10-11   |   分类于 daily   |  

Without having to write any custom code to specific problem. Just feed data to the generic algorithm and it builds its own logical base on the data.

For instance, there is a kind of algorithm named classification. You can feed handwritten letters to the generic algorithm, then it can put data into different groups, here are letters. Also we could feed the generic algorithm with spam or - not-spam emails, then it will put them into two groups: spam and not-spam.

“Machine Learning” includes different kinds of generic algorithms like classification algorithm. It can split into two main categories - supervised learning and unsupervised learning.
